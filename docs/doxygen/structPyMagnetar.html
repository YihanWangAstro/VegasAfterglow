Detailed Description

Magnetar model for gamma-ray burst central engine energy injection.

This class represents a magnetar spin-down model with characteristic luminosity, spin-down time scale, and power-law index. The magnetar provides continuous energy injection into the gamma-ray burst ejecta, following the relation: L(t) = L0 * (1 + t/t0)^(-q). This model is commonly used to explain X-ray plateaus and late-time energy injection in gamma-ray bursts.
