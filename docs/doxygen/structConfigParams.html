Detailed Description

Configuration parameters for numerical setup and advanced physics in afterglow modeling.

This structure controls the numerical resolution, cosmological setup, and optional advanced physics processes in gamma-ray burst afterglow calculations. These parameters determine the computational accuracy, efficiency, and the physical processes included in the simulation without affecting the fundamental model parameters.
