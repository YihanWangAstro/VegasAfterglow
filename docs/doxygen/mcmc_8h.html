PyOutput

Container for MCMC simulation output data as key-value pairs.

This type provides a convenient mapping from string identifiers to numerical arrays containing MCMC results such as parameter chains, likelihood values, acceptance rates, and other diagnostic quantities. The use of xt::xarray enables efficient storage and manipulation of multi-dimensional output data.
