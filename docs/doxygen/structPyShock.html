Detailed Description

Comprehensive shock evolution data for detailed afterglow analysis.

This structure contains all the physical quantities that evolve during shock propagation, including dynamics, magnetic fields, particle distributions, and characteristic frequencies. It provides complete information for understanding the physical processes driving the afterglow emission.
