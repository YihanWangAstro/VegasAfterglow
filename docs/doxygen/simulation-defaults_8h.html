Detailed Description

Centralized simulation configuration defaults.

This header consolidates all tunable simulation parameters in one place. These are numerical/algorithmic parameters, NOT physical constants.

Benefits:

  • Single source of truth for all defaults
  • Easy to create different presets (fast vs. accurate)
  • Clear documentation of what each parameter does
  • No magic numbers scattered across the codebase
