Detailed Description

template<typename Ejecta, typename Medium>
struct ForwardState< Ejecta, Medium >

State vector structure for forward shock calculations.

Template structure that defines the state vector for forward shock calculations, adapting its size based on whether the Ejecta class supports mass and energy injection methods.
