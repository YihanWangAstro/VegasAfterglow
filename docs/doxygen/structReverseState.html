Detailed Description

template<typename Ejecta, typename Medium>
struct ReverseState< Ejecta, Medium >

Represents the state variables for the reverse shock simulation.

It defines a state vector containing properties like shell width, mass, radius, time, and energy. The struct dynamically adapts its size based on template parameters to include mass/energy injection capabilities.
