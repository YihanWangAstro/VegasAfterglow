Computes the downstream four-velocity for a given relative Lorentz factor and magnetization parameter.

Parameters
gamma_relRelative Lorentz factor between upstream and downstream regions
sigmaMagnetization parameter (ratio of magnetic to rest-mass energy density)
Returns
The downstream four-velocity in the shock frame

The calculation handles both magnetized (sigma > 0) and non-magnetized (sigma = 0) cases using different equations based on jump conditions across the shock front.
