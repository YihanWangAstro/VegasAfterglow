Smooth broken power-law transition in log2 space.

Returns a correction term that smoothly changes the slope by delta_beta at the break. Far below break: correction -> 0 (original slope preserved). Far above break: correction -> -delta_beta * (log2_x - log2_x_break) / s (slope shifted).

Parameters
log2_xLog2 of the evaluation point
log2_x_breakLog2 of the break point
s_delta_betas * (slope_below - slope_above), where s controls transition sharpness

Returns
Log2-space correction to add to the base power law
