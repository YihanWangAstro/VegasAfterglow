void broadcast_symmetry ( Array &  arr,
Coord const &  coord 
)

Broadcasts computed values across grid based on coord symmetry.

Template Parameters
Array3D array type (e.g., ICPhotonGrid or ElectronGrid)
Parameters
arrThe 3D array to broadcast
coordCoordinate object containing symmetry information and theta representatives

This function handles the broadcasting of computed values from representative grid points to all grid points based on the detected symmetry level:

  • isotropic: Broadcasts from (0,0) to all (phi, theta)
  • piecewise: Broadcasts within theta groups, then across phi
  • phi_symmetric: Broadcasts only across phi direction
