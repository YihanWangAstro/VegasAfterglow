Detailed Description

Complete parameter set for gamma-ray burst afterglow model configuration.

This structure encapsulates all physical and observational parameters needed to specify a gamma-ray burst afterglow model, including jet geometry, energy distribution, external medium properties, and radiation physics. These parameters serve as the variables for MCMC sampling and likelihood evaluation in parameter estimation studies.
