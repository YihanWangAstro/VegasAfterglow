Detailed Description

Defines a grid for numerical integration in log-space.

Stack struct to avoid memory allocation.

Given minimum and maximum values for nu and gamma, it computes logarithmically spaced bins (nu_edge and gamma_edge) and then determines center values (nu and gamma) from those bins.
